# Moonboy — Tech Blog

A personal tech blog built with Next.js 14. Writing about AI, full-stack engineering, and the real process behind shipping software.

## Live Demo

[moonboyblog.vercel.app](https://moonboyblog.vercel.app)

## Tech Stack

- Next.js 14 (App Router)
- React 19
- CSS Modules
- MongoDB (coming Week 2)
- Auth.js (coming Week 3)

## Pages

- `/` — Homepage with hero and featured posts
- `/about` — About page with stats
- `/contact` — Contact form
- `/blog` — Blog listing
- `/blog/[slug]` — Single post page

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

Copy `.env.example` to `.env` and fill in the values:

```bash
cp .env.example .env
```

## Deployment

Deployed on Vercel. Auto-deploys from `main` branch.
